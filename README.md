# Figuras_geometricas-
Este taller tiene como objetivo aplicar los principios fundamentales de la Programación Orientada a Objetos (POO) en Python mediante la implementación de una jerarquía de clases que representa figuras geométricas. Se trabajan conceptos como encapsulamiento, herencia, sobrescritura de métodos y polimorfismo, siguiendo el estándar de escritura PEP8.
